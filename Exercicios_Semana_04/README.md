# Q2)
- Ocorre erro de execução pois é preciso instanciar um objeto do tipo *Conta* antes de acrescentá-lo ao *array*.
- Solução: 
```Java
	Conta[] contas = new Conta[5];
	Conta c1 = new Conta("1", 10);
	contas[0] = c1;
	System.out.println(contas[0].saldo);
```
